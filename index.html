Privacy Policy

Application: InstruCal Pro

Developer: Tattaz Tech

Data Collection

InstruCal Pro stores data locally on the user's device. The application may store calibration information, client information, photos, signatures and generated PDF certificates.

Data Sharing

We do not sell, share or transfer user data to third parties. No data is sent to a server, cloud service or external database.

Data Storage

All data remains on the user's device unless the user chooses to export or share files manually.

Permissions

The application may use device permissions such as camera or storage only to allow users to add photos, signatures and generate certificates.

Data Deletion

Users can delete their data by deleting the generated files or uninstalling the application.
